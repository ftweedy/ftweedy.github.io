TrustFlag


NGO Flagging System
A program that was the result of the 4th Annual San Diego Zoohackathon. I developed the Javascript/React front-end. The project is a way for conservation NGOs(Non-Governmental Organizations) to anonymously coordinate operations by placing flags on their targets. If a different user seraches the system, a positive result returns only the contact information of the flag owner.
